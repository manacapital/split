Como instalar o aplicativo

Android

  1. Abrir o link do APP no navegador Google Chrome
  2. Tocar no menu ⋮ (três pontinhos)
  3. Selecionar Adicionar à tela inicial ou Instalar aplicativo
  4. O sistema aparecerá como um aplicativo no celular

iPhone (iOS)

  1. Abrir o link do APP no navegador Safari
  2. Tocar no botão Compartilhar
  3. Selecionar Adicionar à Tela de Início
  4. O aplicativo aparecerá como um APP no iPhone
